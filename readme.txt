git add -A подготовить все измнения к запоминания состояние(флаг -A)
git commit -m "comment message" запомнить состояние(но только то что было добавлено комадной git add )
git push origin master отправить изменения в удаленную репозиторий
git pull origin master скачать изменения с удаленного репозитория
git remote add origin https:// ссылка которую даст github
